# Обчислення визначеного інтеграла методом Монте-Карло

У даній роботі було реалізовано чисельний алгоритм обчислення визначеного інтеграла
функції f(x) = x² на інтервалі [0, 2] за допомогою методу Монте-Карло.

Для перевірки коректності реалізації отриманий результат було порівняно
з аналітичним значенням інтеграла та результатом, отриманим за допомогою
функції quad з модуля scipy.integrate.

Аналітичне значення інтеграла:
∫₀² x² dx = 8/3 ≈ 2.6667

Результат, отриманий методом Монте-Карло, є близьким до точного значення,
а похибка зменшується зі збільшенням кількості випадкових точок.
Це підтверджує правильність реалізації алгоритму.

Метод Монте-Карло є простим і універсальним чисельним методом,
який особливо ефективний для складних або багатовимірних інтегралів.
